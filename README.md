# StudentManagmentSystem
A simple Student Management System built using Java and MySQL.
This project helps to add, view, update, delete, and search student records efficiently.
All student data is stored securely in a MySQL database, making the system reliable and scalable.

The project is menu-driven and easy to understand, making it ideal for students and beginners to learn Java database connectivity (JDBC) and core Java concepts.
